SGnre
=================

Objetivo
-----
Essa lib é para ser um forma simples de se gerar o XML de um Lote para processamentos das GNRE no site http://www.gnre.pe.gov.br/

Dependências
-------
* [Apache](http://httpd.apache.org/) / [Nginx](http://nginx.org/)
* [PHP 5.3+](http://php.net)
* Extensões PHP
 * [DOMDocument](http://php.net/manual/pt_BR/class.domdocument.php)

Documentação
-----
Baseado no manual disponibilizado pelos site da GNRE http://www.gnre.pe.gov.br/

Referencia
-----
